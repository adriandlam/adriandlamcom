## Combined Example

Here's a realistic example combining multiple features:

The time complexity of binary search is $O(\log n)$, which makes it much faster than linear search with $O(n)$ complexity. Here's an implementation:

```javascript
function binarySearch(arr, target) {
  let left = 0;
  let right = arr.length - 1;
  
  while (left <= right) {
    const mid = Math.floor((left + right) / 2);
    
    if (arr[mid] === target) {
      return mid;  // Found the target!
    } else if (arr[mid] < target) {
      left = mid + 1;  // Search right half
    } else {
      right = mid - 1;  // Search left half
    }
  }
  
  return -1;  // Target not found
}

// Test it out
const numbers = [1, 3, 5, 7, 9, 11, 13, 15];
console.log(binarySearch(numbers, 7));  // Output: 3
```

You can learn more about algorithms on [Wikipedia](https://en.wikipedia.org/wiki/Binary_search_algorithm) or in [Introduction to Algorithms](https://mitpress.mit.edu/9780262046305/introduction-to-algorithms/).
